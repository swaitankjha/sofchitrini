# 📱 Chitrini – Privacy-First Chat & Video Calling App

Chitrini is a modern Android messaging application focused on **privacy, reliability, and performance**. It combines **Socket.IO**, **Firebase**, and **WebRTC** to deliver fast real-time messaging, secure media sharing, and high-quality video calling while remaining lightweight enough for lower-end Android devices.

---

## ✨ Features

### 🔒 Privacy & Security
- End-to-End Encryption (E2EE) for secure conversations
- Privacy-first architecture with **one device = one unique username**
- Smart notification system designed to protect user privacy
- No unnecessary user data collection

### 💬 Messaging
- Real-time messaging powered by **Socket.IO**
- Dual-channel message delivery using **Socket.IO + Firebase** for improved reliability
- Swipe-to-reply support
- Media sharing (images, videos, and files)
- Fast and responsive chat experience

### 📹 Video Calling
- High-quality video calls using **WebRTC**
- Dual-channel signaling architecture for smoother and more stable connections
- Optimized for low-latency communication

### 🎨 User Experience
- Profile picture editing
- Daily calming Sanskrit quote
- Clean and modern Material Design interface
- Lightweight UI optimized for smooth navigation

### ⚡ Performance
- Optimized graphics rendering for better compatibility with entry-level and older Android devices
- Reduced hardware resource consumption
- Efficient memory management for smoother multitasking
- Battery-conscious background operations

---

## 🛠 Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** MVVM
- **Realtime Messaging:** Socket.IO
- **Cloud Services:** Firebase
- **Video Calling:** WebRTC
- **Database:** Firebase Firestore
- **Authentication:** Anonymous Device-Based Identity
